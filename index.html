    • Dart Web Toolkit

    Dart Web Toolkit (DWT) is a development toolkit for building and optimizing complex browser-based applications inspired by GWT.

    Project maintained by akserg Hosted on GitHub Pages — Theme by mattgraham

    What is the DWT?

    DWT is a port of GWT written in pure Dart language. You can create a new enterprise level web application fairly quickly with the DWT. DWT has a rich set of widgets, panels and utility classes compatible with GWT, so migration from an existing Java web applications to the Dart will be faster and easier. You can use your existing GWT themes in the DWT.

    What under the bonnet?

    DWT is a set of libraries:

    • Activity library helps organise navigation applications with activities;
    • Animation library provides animation support for variety of the DWT components;
    • Core library contains a small number of core classes;
    • Data library contains classes for working with the data;
    • Editor library helps editing bean-like objects;
    • Event library sharing both browser and non-browser events, adapters and event handlers;
    • Http library helps to build HTTP requests;
    • I18n library helps with the internalisation, text alignment and direction;
    • Layout library knows how laying out a container elements and its children;
    • Place library contains classes for working with the bookmark location in the web application;
    • Resource library helps to provide access to resources at runtime;
    • Role library allows ARIA Accessibility attributes to be added to widgets so that they can be identified by assistive technology;
    • Scheduler library provides low-level task scheduling primitives;
    • Text library helps organise rendering objects into plain-text form;
    • UI library is user-interface objects;
    • Util library is a small set of utility classes;

    How to create a brand new web application on the DWT?

    Note: If you are just beginning to study the Dart language please read the tutorial first. You need to have latest version of the DartEditor with the Dart SDK installed on your computer.

    • Create a new web application hello_world.
    • Open pubspec.yaml and add the Dart Web Toolkit into Dependencies:
    dependencies:
  dart_web_toolkit: ">=0.3.0"
    • Save pubspec.yaml and the DartEditor automatically runs pub install, which recursively installs the Dart libraries from the packages in the dependencies list.
    • Open hello_world.html file from the web folder and add a link reference to the Dart Web Toolkit clean theme.
    <link rel="stylesheet" 
    href="packages/dart_web_toolkit/resource/clean/clean.css">
    • Open hello_world.dart and modify it as follows:
    library dwt_hello_world;

import 'package:dart_web_toolkit/ui.dart' as ui;

void main() {
    ui.Label label = new ui.Label("Hello, World!");
    ui.RootLayoutPanel.get().add(label);
}
    • Check how the application works in Dartium.
    • Check how the application works in JavaScript within FireFox or IE.

    Congratulations you have just created your first application on the DWT!
